    ADK лҸ„кө¬м—җ лҢҖн•ң мһ‘м—… нҷ•мқё л°ӣкё°¶

    ADKм—җм„ң м§ҖмӣҗPython v1.14.0мӢӨн—ҳм Ғ

    мқјл¶Җ м—җмқҙм „нҠё мӣҢнҒ¬н”ҢлЎңлҠ” мқҳмӮ¬ кІ°м •, нҷ•мқё, ліҙм•Ҳ лҳҗлҠ” мқјл°ҳм Ғмқё к°җлҸ…мқ„ мң„н•ҙ нҷ•мқёмқҙ н•„мҡ”н•©лӢҲлӢӨ. мқҙлҹ¬н•ң кІҪмҡ° мӣҢнҒ¬н”ҢлЎңлҘј м§„н–үн•ҳкё° м „м—җ мӮ¬лһҢмқҙлӮҳ к°җлҸ… мӢңмҠӨн…ңмңјлЎңл¶Җн„° мқ‘лӢөмқ„ л°ӣм•„м•ј н•©лӢҲлӢӨ. м—җмқҙм „нҠё к°ңл°ң нӮӨнҠё(ADK)мқҳ лҸ„кө¬ нҷ•мқё кё°лҠҘмқ„ мӮ¬мҡ©н•ҳл©ҙ ADK лҸ„кө¬к°Җ мӢӨн–үмқ„ мқјмӢң мӨ‘м§Җн•ҳкі  мӮ¬мҡ©мһҗ лҳҗлҠ” лӢӨлҘё мӢңмҠӨн…ңкіј мғҒнҳё мһ‘мҡ©н•ҳм—¬ нҷ•мқён•ҳкұ°лӮҳ м§„н–үн•ҳкё° м „м—җ кө¬мЎ°нҷ”лҗң лҚ°мқҙн„°лҘј мҲҳм§‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ. лӢӨмқҢкіј к°ҷмқҖ л°©лІ•мңјлЎң ADK лҸ„кө¬мҷҖ н•Ёк»ҳ лҸ„кө¬ нҷ•мқёмқ„ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • л¶Җмҡё нҷ•мқё: require_confirmation л§Өк°ңліҖмҲҳлҘј мӮ¬мҡ©н•ҳм—¬ FunctionToolмқ„ кө¬м„ұн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ мҳөм…ҳмқҖ мҳҲ лҳҗлҠ” м•„лӢҲмҡ” нҷ•мқё мқ‘лӢөмқ„ мң„н•ҙ лҸ„кө¬лҘј мқјмӢң мӨ‘м§Җн•©лӢҲлӢӨ.
    • кі кёү нҷ•мқё: кө¬мЎ°нҷ”лҗң лҚ°мқҙн„° мқ‘лӢөмқҙ н•„мҡ”н•ң мӢңлӮҳлҰ¬мҳӨмқҳ кІҪмҡ° нҷ•мқёмқ„ м„ӨлӘ…н•ҳлҠ” н…ҚмҠӨнҠё н”„лЎ¬н”„нҠёмҷҖ мҳҲмғҒ мқ‘лӢөмңјлЎң FunctionToolмқ„ кө¬м„ұн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    мӢӨн—ҳм Ғ

    лҸ„кө¬ нҷ•мқё кё°лҠҘмқҖ мӢӨн—ҳм Ғмқҙл©° лӘҮ к°Җм§Җ м•Ңл Өм§„ м ңн•ң мӮ¬н•ӯмқҙ мһҲмҠөлӢҲлӢӨ. н”јл“ңл°ұмқ„ нҷҳмҳҒн•©лӢҲлӢӨ!

    мҡ”мІӯмқҙ мӮ¬мҡ©мһҗм—җкІҢ м „лӢ¬лҗҳлҠ” л°©мӢқмқ„ кө¬м„ұн•  мҲҳ мһҲмңјл©° мӢңмҠӨн…ңмқҖ ADK м„ңлІ„мқҳ REST APIлҘј нҶөн•ҙ м „мҶЎлҗң мӣҗкІ© мқ‘лӢөмқ„ мӮ¬мҡ©н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. ADK мӣ№ мӮ¬мҡ©мһҗ мқён„°нҺҳмқҙмҠӨмҷҖ н•Ёк»ҳ нҷ•мқё кё°лҠҘмқ„ мӮ¬мҡ©н•ҳлҠ” кІҪмҡ° м—җмқҙм „нҠё мӣҢнҒ¬н”ҢлЎңлҠ” к·ёлҰј 1кіј к°ҷмқҙ мӮ¬мҡ©мһҗм—җкІҢ лҢҖнҷ” мғҒмһҗлҘј н‘ңмӢңн•ҳм—¬ мһ…л Ҙмқ„ мҡ”мІӯн•©лӢҲлӢӨ.

    к·ёлҰј 1. кі кёү лҸ„кө¬ мқ‘лӢө кө¬нҳ„мқ„ мӮ¬мҡ©н•ҳлҠ” мҳҲм ң нҷ•мқё мқ‘лӢө мҡ”мІӯ лҢҖнҷ” мғҒмһҗ.

    лӢӨмқҢ м„№м…ҳм—җм„ңлҠ” нҷ•мқё мӢңлӮҳлҰ¬мҳӨм—җ мқҙ кё°лҠҘмқ„ мӮ¬мҡ©н•ҳлҠ” л°©лІ•мқ„ м„ӨлӘ…н•©лӢҲлӢӨ. м „мІҙ мҪ”л“ң мғҳн”ҢмқҖ human_tool_confirmation мҳҲлҘј м°ёмЎ°н•ҳмӢӯмӢңмҳӨ. м—җмқҙм „нҠё мӣҢнҒ¬н”ҢлЎңм—җ мӮ¬лһҢмқҳ мһ…л Ҙмқ„ нҶөн•©н•ҳлҠ” м¶”к°Җ л°©лІ•мқҙ мһҲмҠөлӢҲлӢӨ. мһҗм„ён•ң лӮҙмҡ©мқҖ Human-in-the-loop м—җмқҙм „нҠё нҢЁн„ҙмқ„ м°ёмЎ°н•ҳмӢӯмӢңмҳӨ.

    л¶Җмҡё нҷ•мқё¶

    лҸ„кө¬м—җ мӮ¬мҡ©мһҗмқҳ к°„лӢЁн•ң мҳҲ лҳҗлҠ” м•„лӢҲмҡ”л§Ң н•„мҡ”н•ң кІҪмҡ° FunctionTool нҒҙлһҳмҠӨлҘј лһҳнҚјлЎң мӮ¬мҡ©н•ҳм—¬ нҷ•мқё лӢЁкі„лҘј м¶”к°Җн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ reimburseлқјлҠ” лҸ„кө¬к°Җ мһҲлҠ” кІҪмҡ° лӢӨмқҢ мҳҲмҷҖ к°ҷмқҙ FunctionTool нҒҙлһҳмҠӨлЎң лһҳн•‘н•ҳкі  require_confirmation л§Өк°ңліҖмҲҳлҘј TrueлЎң м„Өм •н•ҳм—¬ нҷ•мқё лӢЁкі„лҘј нҷңм„ұнҷ”н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    # agent.pyм—җм„ң
root_agent = Agent(
   ...
   tools=[
        # лҸ„кө¬ нҳём¶ңм—җ лҢҖн•ң мӮ¬мҡ©мһҗ нҷ•мқёмқ„ мҡ”кө¬н•ҳл Өл©ҙ require_confirmationмқ„ TrueлЎң м„Өм •н•©лӢҲлӢӨ.
        FunctionTool(reimburse, require_confirmation=True),
    ],
...

    мқҙ кө¬нҳ„ л°©лІ•мқҖ мөңмҶҢн•ңмқҳ мҪ”л“ңк°Җ н•„мҡ”н•ҳм§Җл§Ң мӮ¬мҡ©мһҗ лҳҗлҠ” нҷ•мқё мӢңмҠӨн…ңмқҳ к°„лӢЁн•ң мҠ№мқёмңјлЎң м ңн•ңлҗ©лӢҲлӢӨ. мқҙ м ‘к·ј л°©мӢқмқҳ м „мІҙ мҳҲлҠ” human_tool_confirmation мҪ”л“ң мғҳн”Ңмқ„ м°ёмЎ°н•ҳмӢӯмӢңмҳӨ.

    нҷ•мқё н•ЁмҲҳ н•„мҡ”¶

    мһ…л Ҙ к°’мқ„ л¶Җмҡё мқ‘лӢөмқ„ л°ҳнҷҳн•ҳлҠ” н•ЁмҲҳлЎң л°”кҫём–ҙ require_confirmation мқ‘лӢөмқҳ лҸҷмһ‘мқ„ мҲҳм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ. лӢӨмқҢ мҳҲлҠ” нҷ•мқёмқҙ н•„мҡ”н•ңм§Җ м—¬л¶ҖлҘј кІ°м •н•ҳлҠ” н•ЁмҲҳлҘј ліҙм—¬мӨҚлӢҲлӢӨ.

    async def confirmation_threshold(
    amount: int, tool_context: ToolContext
) -> bool:
  """кёҲм•Ўмқҙ 1000ліҙлӢӨ нҒ¬л©ҙ trueлҘј л°ҳнҷҳн•©лӢҲлӢӨ."""
  return amount > 1000

    мқҙ н•ЁмҲҳлҠ” require_confirmation л§Өк°ңліҖмҲҳмқҳ л§Өк°ңліҖмҲҳ к°’мңјлЎң м„Өм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    root_agent = Agent(
   ...
   tools=[
        # мӮ¬мҡ©мһҗ нҷ•мқёмқ„ мҡ”кө¬н•ҳл Өл©ҙ require_confirmationмқ„ TrueлЎң м„Өм •н•©лӢҲлӢӨ.
        FunctionTool(reimburse, require_confirmation=confirmation_threshold),
    ],
...

    мқҙ кө¬нҳ„мқҳ м „мІҙ мҳҲлҠ” human_tool_confirmation мҪ”л“ң мғҳн”Ңмқ„ м°ёмЎ°н•ҳмӢӯмӢңмҳӨ.

    кі кёү нҷ•мқё¶

    лҸ„кө¬ нҷ•мқём—җ мӮ¬мҡ©мһҗм—җ лҢҖн•ң мһҗм„ён•ң лӮҙмҡ©мқҙлӮҳ лҚ” ліөмһЎн•ң мқ‘лӢөмқҙ н•„мҡ”н•ң кІҪмҡ° tool_confirmation кө¬нҳ„мқ„ мӮ¬мҡ©н•ҳмӢӯмӢңмҳӨ. мқҙ м ‘к·ј л°©мӢқмқҖ ToolContext к°ңмІҙлҘј нҷ•мһҘн•ҳм—¬ мӮ¬мҡ©мһҗм—җ лҢҖн•ң мҡ”мІӯм—җ лҢҖн•ң н…ҚмҠӨнҠё м„ӨлӘ…мқ„ м¶”к°Җн•ҳкі  лҚ” ліөмһЎн•ң мқ‘лӢө лҚ°мқҙн„°лҘј н—Ҳмҡ©н•©лӢҲлӢӨ. мқҙ л°©лІ•мңјлЎң лҸ„кө¬ нҷ•мқёмқ„ кө¬нҳ„н•  л•Ң лҸ„кө¬ мӢӨн–үмқ„ мқјмӢң мӨ‘м§Җн•ҳкі  нҠ№м • м •ліҙлҘј мҡ”мІӯн•ң лӢӨмқҢ м ңкіөлҗң лҚ°мқҙн„°лЎң лҸ„кө¬лҘј лӢӨмӢң мӢңмһ‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    мқҙ нҷ•мқё нқҗлҰ„м—җлҠ” мӢңмҠӨн…ңмқҙ мӮ¬лһҢмқҳ мқ‘лӢөм—җ лҢҖн•ң мһ…л Ҙ мҡ”мІӯмқ„ мЎ°н•©н•ҳм—¬ ліҙлӮҙлҠ” мҡ”мІӯ лӢЁкі„мҷҖ мӢңмҠӨн…ңмқҙ л°ҳнҷҳлҗң лҚ°мқҙн„°лҘј мҲҳмӢ н•ҳкі  мІҳлҰ¬н•ҳлҠ” мқ‘лӢө лӢЁкі„к°Җ мһҲмҠөлӢҲлӢӨ.

    нҷ•мқё м •мқҳ¶

    кі кёү нҷ•мқёмңјлЎң лҸ„кө¬лҘј л§Ңл“Ө л•Ң ToolContext к°ңмІҙлҘј нҸ¬н•Ён•ҳлҠ” н•ЁмҲҳлҘј л§Ңл“ӯлӢҲлӢӨ. к·ёлҹ° лӢӨмқҢ tool_confirmation к°ңмІҙ, hint л°Ҹ payload л§Өк°ңліҖмҲҳк°Җ мһҲлҠ” tool_context.request_confirmation() л©”м„ңл“ңлҘј мӮ¬мҡ©н•ҳм—¬ нҷ•мқёмқ„ м •мқҳн•©лӢҲлӢӨ. мқҙлҹ¬н•ң мҶҚм„ұмқҖ лӢӨмқҢкіј к°ҷмқҙ мӮ¬мҡ©лҗ©лӢҲлӢӨ.

    • hint: мӮ¬мҡ©мһҗм—җкІҢ н•„мҡ”н•ң кІғмқ„ м„ӨлӘ…н•ҳлҠ” м„ӨлӘ… л©”мӢңм§Җмһ…лӢҲлӢӨ.
    • payload: л°ҳнҷҳлҗ  кІғмңјлЎң мҳҲмғҒлҗҳлҠ” лҚ°мқҙн„°мқҳ кө¬мЎ°мһ…лӢҲлӢӨ. мқҙ лҚ°мқҙн„° мң нҳ•мқҖ Anyмқҙл©° мӮ¬м „ лҳҗлҠ” pydantic лӘЁлҚёкіј к°ҷмқҙ JSON нҳ•мӢқ л¬ёмһҗм—ҙлЎң м§Ғл ¬нҷ”н•  мҲҳ мһҲм–ҙм•ј н•©лӢҲлӢӨ.

    лӢӨмқҢ мҪ”л“ңлҠ” м§Ғмӣҗмқҳ нңҙк°Җ мҡ”мІӯмқ„ мІҳлҰ¬н•ҳлҠ” лҸ„кө¬м—җ лҢҖн•ң мҳҲм ң кө¬нҳ„мқ„ ліҙм—¬мӨҚлӢҲлӢӨ.

    return {'status': 'кҙҖлҰ¬мһҗ мҠ№мқёмқҙ н•„мҡ”н•©лӢҲлӢӨ.'}

      'status': 'ok',

    мқҙ м ‘к·ј л°©мӢқмқҳ м „мІҙ мҳҲлҠ” human_tool_confirmation мҪ”л“ң мғҳн”Ңмқ„ м°ёмЎ°н•ҳмӢӯмӢңмҳӨ. нҷ•мқёмқҙ м§„н–үлҗҳлҠ” лҸҷм•Ҳ м—җмқҙм „нҠё мӣҢнҒ¬н”ҢлЎң лҸ„кө¬ мӢӨн–үмқҙ мқјмӢң мӨ‘м§ҖлҗңлӢӨлҠ” м җмқ„ лӘ…мӢ¬н•ҳмӢӯмӢңмҳӨ. нҷ•мқёмқҙ мҲҳмӢ лҗң нӣ„ tool_confirmation.payload к°ңмІҙм—җм„ң нҷ•мқё мқ‘лӢөм—җ м•Ўм„ёмҠӨн•ң лӢӨмқҢ мӣҢнҒ¬н”ҢлЎң мӢӨн–үмқ„ кі„мҶҚн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    REST APIлҘј мӮ¬мҡ©н•ң мӣҗкІ© нҷ•мқё¶

    м—җмқҙм „нҠё мӣҢнҒ¬н”ҢлЎңм—җ лҢҖн•ң мӮ¬лһҢмқҳ нҷ•мқёмқ„ мң„н•ң нҷңм„ұ мӮ¬мҡ©мһҗ мқён„°нҺҳмқҙмҠӨк°Җ м—ҶлҠ” кІҪмҡ° лӘ…л №мӨ„ мқён„°нҺҳмқҙмҠӨлҘј нҶөн•ҙ нҷ•мқёмқ„ мІҳлҰ¬н•ҳкұ°лӮҳ мқҙл©”мқј лҳҗлҠ” мұ„нҢ… м• н”ҢлҰ¬мјҖмқҙм…ҳкіј к°ҷмқҖ лӢӨлҘё мұ„л„җмқ„ нҶөн•ҙ лқјмҡ°нҢ…н•  мҲҳ мһҲмҠөлӢҲлӢӨ. лҸ„кө¬ нҳём¶ңмқ„ нҷ•мқён•ҳл Өл©ҙ мӮ¬мҡ©мһҗ лҳҗлҠ” нҳём¶ң м• н”ҢлҰ¬мјҖмқҙм…ҳмқҙ лҸ„кө¬ нҷ•мқё лҚ°мқҙн„°мҷҖ н•Ёк»ҳ FunctionResponse мқҙлІӨнҠёлҘј ліҙлӮҙм•ј н•©лӢҲлӢӨ.

    ADK API м„ңлІ„мқҳ /run лҳҗлҠ” /run_sse м—”л“ңнҸ¬мқёнҠё лҳҗлҠ” ADK мӢӨн–үкё°лЎң м§Ғм ‘ мҡ”мІӯмқ„ ліҙлӮј мҲҳ мһҲмҠөлӢҲлӢӨ. лӢӨмқҢ мҳҲм—җм„ңлҠ” curl лӘ…л №мқ„ мӮ¬мҡ©н•ҳм—¬ /run_sse м—”л“ңнҸ¬мқёнҠёлЎң нҷ•мқёмқ„ ліҙлғ…лӢҲлӢӨ.

    нҷ•мқём—җ лҢҖн•ң REST кё°л°ҳ мқ‘лӢөмқҖ лӢӨмқҢ мҡ”кө¬ мӮ¬н•ӯмқ„ м¶©мЎұн•ҙм•ј н•©лӢҲлӢӨ.

    • response к°ңмІҙм—җлҠ” нҷ•мқё мғҒнғңмҷҖ лҸ„кө¬м—җ н•„мҡ”н•ң м¶”к°Җ нҺҳмқҙлЎңл“ң лҚ°мқҙн„°к°Җ нҸ¬н•Ёлҗ©лӢҲлӢӨ.

    м°ёкі : мһ¬к°ң кё°лҠҘмңјлЎң нҷ•мқё

    ADK м—җмқҙм „нҠё мӣҢнҒ¬н”ҢлЎңк°Җ мһ¬к°ң кё°лҠҘмңјлЎң кө¬м„ұлҗң кІҪмҡ° нҷ•мқё мқ‘лӢөкіј н•Ёк»ҳ нҳём¶ң ID(invocation_id) л§Өк°ңліҖмҲҳлҸ„ нҸ¬н•Ён•ҙм•ј н•©лӢҲлӢӨ. м ңкіөн•ҳлҠ” нҳём¶ң IDлҠ” нҷ•мқё мҡ”мІӯмқ„ мғқм„ұн•ң кІғкіј лҸҷмқјн•ң нҳём¶ңмқҙм–ҙм•ј н•©лӢҲлӢӨ. к·ёл Үм§Җ м•Ҡмңјл©ҙ мӢңмҠӨн…ңмқҙ нҷ•мқё мқ‘лӢөмңјлЎң мғҲ нҳём¶ңмқ„ мӢңмһ‘н•©лӢҲлӢӨ. м—җмқҙм „нҠёк°Җ мһ¬к°ң кё°лҠҘмқ„ мӮ¬мҡ©н•ҳлҠ” кІҪмҡ° мқ‘лӢөм—җ нҸ¬н•Ёлҗ  мҲҳ мһҲлҸ„лЎқ нҷ•мқё мҡ”мІӯкіј н•Ёк»ҳ нҳём¶ң IDлҘј л§Өк°ңліҖмҲҳлЎң нҸ¬н•Ён•ҳлҠ” кІғмқ„ кі л Өн•ҳмӢӯмӢңмҳӨ. мһ¬к°ң кё°лҠҘ мӮ¬мҡ©м—җ лҢҖн•ң мһҗм„ён•ң лӮҙмҡ©мқҖ мӨ‘м§Җлҗң м—җмқҙм „нҠё мһ¬к°ңлҘј м°ёмЎ°н•ҳмӢӯмӢңмҳӨ.

    м•Ңл Өм§„ м ңн•ң мӮ¬н•ӯ¶

    лҸ„кө¬ нҷ•мқё кё°лҠҘм—җлҠ” лӢӨмқҢкіј к°ҷмқҖ м ңн•ң мӮ¬н•ӯмқҙ мһҲмҠөлӢҲлӢӨ.

    • DatabaseSessionServiceлҠ” мқҙ кё°лҠҘм—җм„ң м§Җмӣҗлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.
    • VertexAiSessionServiceлҠ” мқҙ кё°лҠҘм—җм„ң м§Җмӣҗлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.
