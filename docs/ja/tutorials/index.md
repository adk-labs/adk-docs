# ADKでエージェントを構築する

エージェント開発キット (ADK) の実践的なガイド集で、さっそく始めましょう。これらのチュートリアルは、ADKのさまざまな機能を紹介するために、シンプルで段階的なステップバイステップ方式で設計されています。

このアプローチにより、基礎的な概念から始め、徐々により高度なエージェント開発テクニックに取り組むことで、段階的に学び、構築することができます。さまざまなユースケースでこれらの機能を効果的に適用する方法を探求し、ADKを使用して独自の高度なエージェントアプリケーションを構築できるようになります。以下のコレクションをご覧になり、ビルドをお楽しみください:
