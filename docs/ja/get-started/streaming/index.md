# ストリーミングエージェントの構築

Agent Development Kit (ADK) は、ストリーミングを通じてAIエージェントとのリアルタイムかつインタラクティブな体験を実現します。これにより、ライブ音声会話、リアルタイムなツール使用、エージェントからの継続的な更新などの機能が可能になります。

このページでは、PythonおよびJava ADKの両方でストリーミング機能をすぐに利用開始するためのクイックスタート例を提供します。
