에이전트 개발 키트(ADK)를 사용하면 [Agent2Agent(A2A) 프로토콜](https://a2a-protocol.org/)을 사용하여 다양한 에이전트가 협력하고 상호 작용해야 하는 복잡한 다중 에이전트 시스템을 구축할 수 있습니다! 이 섹션에서는 에이전트가 안전하고 효율적으로 통신하고 협업할 수 있는 강력한 다중 에이전트 시스템을 구축하기 위한 포괄적인 가이드를 제공합니다.

ADK의 A2A 기능에 대해 알아보려면 아래 가이드를 참조하세요.

  **[A2A 소개](./intro.md)**

  루트 에이전트, 로컬 하위 에이전트 및 원격 A2A 에이전트로 다중 에이전트 시스템을 구축하여 A2A의 기본 사항을 배우려면 여기에서 시작하세요. 다음 가이드에서는 다른 에이전트가 A2A 프로토콜을 통해 에이전트를 사용할 수 있도록 에이전트를 노출하는 방법을 다룹니다.

  *   **[A2A 빠른 시작(노출) - Go](./quickstart-exposing-go.md)**

  이 가이드에서는 에이전트가 A2A 프로토콜을 사용하여 다른 원격 에이전트를 사용하도록 허용하는 방법을 보여줍니다.
